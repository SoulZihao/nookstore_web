# NookStore - Organize Every Nook

> **Small Spaces, Big Potential.**
> NookStore 致力于通过精巧的收纳与家具设计，激发家居空间的无限可能。

---

## 📂 项目结构 (Project Structure)

本仓库包含 NookStore 独立站的前端源代码及品牌资产：

* **HTML 页面**: `index.html` (首页), `product.html` (产品页), `about.html` (关于我们), `contact.html` (联系我们)。
* **样式/脚本**: `scss/` & `css/` (样式表), `js/` (交互逻辑)。
* **资源文件**: `assets/` (存放 Logo、产品图片等媒体资源)。
***其他**: `一级目录列表.txt` (产品分类索引), `foreign_website.code-workspace` (VS Code 工作区配置)。

---

## 📦 产品系列 (Product Categories)

根据项目资料，我们的核心产品线包括：

### 1. 实用家具 (Stools & Seating)
***条纹与功能系列**: 2059 条纹凳、2105 多功能楼梯凳。
***座椅系列**: 2125 靠背椅、2144-2146 凳子及软包凳系列。

### 2. 全屋收纳方案 (Home Organization)
***大容量存储**: 6212-6236 折叠收纳箱、侧开门收纳箱及大力士系列。
***精细化收纳**: 化妆盒、证件文件收纳、透明收纳盒及桌面收纳。
***生活场景**: 卫生间收纳、床底收纳、衣橱收纳橱子。

### 3. 厨房与保鲜 (Kitchen & Food Storage)
***食材管理**: 冰箱收纳盒、饺子盒、玻璃保鲜盒及保鲜盒系列。
***烹饪辅助**: 多功能撒盐罐、油壶及厨房夹缝收纳盒。

## 🚀 快速开始

1.  **克隆仓库**
    ```bash
    git clone [https://github.com/SoulZihao/nookstore_web.git](https://github.com/SoulZihao/nookstore_web.git)
    ```
2.  **本地预览**
    直接在浏览器中打开 `index.html` 或通过 VS Code 的 Live Server 扩展启动。
3.  **查看产品明细**
    请参阅 `一级目录列表.txt` 获取 2000+ 条产品分类编号。
